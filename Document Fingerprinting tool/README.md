# Document Fingerprinting tool

## Problem

This covers UN SDG 9 (BUILD RESILIENT INFRASTRUCTURE, PROMOTE INCLUSIVE AND SUSTAINABLE INDUSTRIALIZATION AND FOSTER INNOVATION.)

Small-to-medium Enterprises (SME) have problems regarding with verifying their files, concerning the security level in it. This tool will solve this problem to ensure that the files are not corrupted or modified, which breaches the integrity side of the security.  

## How it works
1. There are 3 options: Check the hash of a document, Register the document hash, and Verify the hash of the document.
2. Run `python3 file.py` to use the program
